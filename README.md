# Event-Autostart-Bot
This is a simple Discord bot designed to automatically start server events when their scheduled start time is reached. It helps keep your events running on time without requiring manual intervention.

The bot is hosted **24/7** on [railway](https://railway.com/), so it can monitor events continuously and ensure they start as planned.

## Features
- Automatically starts server events at their scheduled times
- Runs continuously in the background
- Minimal setup required, can be added to any server with the proper permissions

## Permissions
To work correctly, the bot needs the following Discord permissions in your server:

- Manage Events
- View Channels (so it can monitor events)

Make sure the bot’s role has these permissions in the server where it will operate.

## Usage
1. Add the bot to your server.
2. Schedule your events as normal in Discord.
3. The bot will automatically start each event at the correct time.

No additional commands are required, the bot works silently in the background.

## Open Use
You are free to use this bot in your own server or modify it as you see fit.
